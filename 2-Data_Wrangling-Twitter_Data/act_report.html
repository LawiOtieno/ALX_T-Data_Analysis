Introduction¶

This project involves all Data Wrangling Processes such as Gathering, Assessing, Cleaning and Storing. Analysis and Visualization are also done.

Three datasets were used, all containing data from a popular Twitter account WeRateDogs.

  • The WeRateDogs Twitter archive. twitter_archive_enhanced.csv

  • The tweet image predictions. image_predictions.tsv

  • Additional data from the Twitter API. tweet_json.txt
